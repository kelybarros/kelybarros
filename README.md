## Sobre Mim

Sou uma **Desenvolvedora Full Stack (Júnior) brasileira**, atuando como **freelancer** no desenvolvimento de aplicações web modernas, escaláveis e preparadas para ambientes de produção.

Desenvolvo soluções completas, desde a criação de **interfaces front-end** até o desenvolvimento de **back-end, APIs e bancos de dados**, sempre priorizando arquitetura organizada, código limpo, desempenho e uma excelente experiência para o usuário.
